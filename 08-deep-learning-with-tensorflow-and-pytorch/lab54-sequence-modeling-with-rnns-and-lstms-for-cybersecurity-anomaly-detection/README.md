# 🧪 Lab 54: Sequence Modeling with RNNs and LSTMs for Cybersecurity Anomaly Detection

## 📌 Lab Overview

This lab focused on understand the fundamentals of recurrent neural networks (rnns) and long short-term memory (lstm) networks through a hands-on Google Colab workflow centered on **Sequence Modeling with RNNs and LSTMs for Cybersecurity Anomaly Detection**.

## 🎯 Objective

- Understand the fundamentals of Recurrent Neural Networks (RNNs) and Long Short-Term Memory (LSTM) networks
- Prepare and preprocess time-series cybersecurity log data for sequence modeling
- Implement tokenization and sequence padding techniques for neural network input
- Build and train an LSTM model for anomaly detection in sequential data
- Evaluate model performance and interpret results for cybersecurity applications
- Apply sequence modeling techniques to identify suspicious patterns in network logs
