# 🧪 Lab 43: Sentiment Analysis on EdTech Feedback

## 📌 Lab Overview

This lab focused on understand the fundamentals of sentiment analysis and its applications in educational technology through a hands-on Google Colab workflow centered on **Sentiment Analysis on EdTech Feedback**.

## 🎯 Objective

- Understand the fundamentals of sentiment analysis and its applications in educational technology
- Load and preprocess labeled feedback datasets for machine learning tasks
- Implement TF-IDF (Term Frequency- Inverse Document Frequency) vectorization for text data
- Build and train a Logistic Regression classifier for sentiment classification
- Evaluate model performance using accuracy, precision, recall, and confusion matrix
- Interpret results and understand the practical implications of sentiment analysis in EdTech
