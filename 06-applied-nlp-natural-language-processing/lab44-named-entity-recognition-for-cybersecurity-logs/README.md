# 🧪 Lab 44: Named Entity Recognition for Cybersecurity Logs

## 📌 Lab Overview

This lab focused on understand the fundamentals of named entity recognition (ner) in cybersecurity contexts through a hands-on Google Colab workflow centered on **Named Entity Recognition for Cybersecurity Logs**.

## 🎯 Objective

- Understand the fundamentals of Named Entity Recognition (NER) in cybersecurity contexts
- Load and process cybersecurity log samples using Python
- Utilize spaCy's NER capabilities to extract IP addresses, organizations, and threat types from security logs
- Visualize named entities using spaCy's built-in visualization tools
- Create custom entity patterns using spaCy's Matcher for specialized cybersecurity terms
- Analyze and interpret NER results for security intelligence purposes
