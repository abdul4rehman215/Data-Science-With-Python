# 🧪 Lab 09: Automate Weather Data Retrieval with API

## 📌 Lab Overview

This lab focused on understand the fundamentals of rest apis and http requests through a hands-on Google Colab workflow centered on **Automate Weather Data Retrieval with API**.

## 🎯 Objective

- Understand the fundamentals of REST APIs and HTTP requests
- Register for and obtain API keys from OpenWeatherMap service
- Make HTTP requests to weather APIs using Python
- Parse JSON responses and extract specific weather data
- Handle API errors and implement basic error checking
- Format and display weather data in a readable tabular format
